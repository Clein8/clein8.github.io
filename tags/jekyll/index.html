정적 사이트 생성기 (Static Site Generator) 비교/선택

인터넷 검색하다 보면 사람들이 ****.github.io라는 주소로 블로깅을 하는걸 보고, 처음에는 Github에서 블로그 서비스를 시작한줄 알았다. Tistory에서 Code Highlighter사용하는게 너무 불편해서, 나도 ****.github.io를 사용해 볼까 하고 검색을 하다 보니 정적 블로그라는 것을 알게 되었다. 정적 사이트 생성기를 이용해서 블로그를 만드는 경우가 많다 보니 정적 블로그라고 불리게 된 것 같다. 그러나 몇개의 사이트 생성기를 사용해 보니 영어를 직역한 표현이긴 해도 정적 사이트 생성기라는 이름이 알맞은 이름 같다. ❏ 정적 사이트 생성기 종류 일단, 선두주자로 제일 오래된 Jekyll이 있다.
