Github Pages에 Hugo 설치하기

❏ Github Pages 언제부턴가 구글에서 코드 검색을 하다 보면 .github.io 사이트를 자주 볼 수 있었다. 잠깐 찾아보니 Github Pages라길래, Github에서 블로그 같은 서비스를 만들었나 싶었다. 그러던 도중 .github.io 사이트들에서 보이는 Code Highlighter가 Tistory에 설치하는 Code Highlighter보다 예뻐 보이고, 코드 입력하기도 더 쉬워 보이길래 좀 더 관심을 가지게 되었다. 며칠을 검색해 보니 코드 저장 해놓기에는 Tistory보다 좋다고 생각되어 시작해 보기로 했다. 그리고, 다 설치하고보니 마크다운(Markdown)으로 글 작성하는게 생각보다 편하고 좋다. ㅎㅎ
